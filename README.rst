Awesome Public Datasets (公共数据集)
=======================

.. image:: https://cdn.rawgit.com/sindresorhus/awesome/d7305f38d29fed78fa85652e3a63e154dd8e8829/media/badge.svg
   :alt: Awesome
   :target: https://github.com/sindresorhus/awesome

这是一个高质量的 `以主题为中心的公共数据源 <https://github.com/awesomedata/awesome-public-datasets>`_ 列表。
它们是从博客、回答和用户响应中收集和整理的。
下面列出的大多数数据集都是免费的，但有些不是。
本项目是在上海交通大学 `OMNILab <https://github.com/OMNILab>`_ 孵化的，期间是 Xiaming Chen 的博士研究阶段。
OMNILab 现在是 `白玉兰开源 AI 社区 <https://github.com/Bai-Yu-Lan>`_ 的一部分。
其他非常棒的列表可以在 `sindresorhus 的 awesome <https://github.com/sindresorhus/awesome>`_ 列表中找到。

**注意**: 此仓库由 `apd-core <https://github.com/awesomedata/apd-core/tree/master/core>`_ 自动生成。
请 **不要** 直接修改此文件。我们提供了一种 `向此仓库贡献 <https://github.com/awesomedata/apd-core/blob/master/CONTRIBUTING.md>`_ 的新方法。
加入 `Slack 社区 <https://awesomedataworld.slack.com>`_ (`点击加入 <https://join.slack.com/t/awesomedataworld/shared_invite/zt-dllew5xy-PJYi~mWUdY3hupohbmVZsA>`_) 以获取高质量数据更新的即时消息。
